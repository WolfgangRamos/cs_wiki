1 Listen

1.1 Einfach verkette Listen

Eine einfach verkette Liste (oder kurz: Liste) besteht aus einer Folge von Knoten. Jeder Knoten enthält hierbei:

  1. ein Datenelement
  2. einen Zeiger auf das nächste Element der Liste, wobei der Zeiger für das letzte Element der Liste auf das null-Element zeigt.

Zusätzlich enthält eine Liste eine Variable head, die auf den ersten Knoten der Liste zeigt, und eine Variable tail, die auf den letzten zeigt. Zeigen sowohl head als auch tail auf das null-Element, so haben wir eine leere Liste.
