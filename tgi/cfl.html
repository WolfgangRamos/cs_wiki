1 Context Free Languages

Eine Sprache \(L\) heißt genau dann kontextfrei (engl. context free language; CFL), wenn es eine CFG \(G\) gibt, die \(L\) erzeugt.

Eine Sprache \(L\) heißt genau dann kontextfrei (engl. context free language; CFL), wenn es einen PDA \(P\) gibt, der \(L\) erkennt.
